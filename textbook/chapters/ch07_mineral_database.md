# 第7章 SLB2011 鉱物データベース

## 7.1 概要

SLB2011 鉱物データベース（Stixrude & Lithgow-Bertelloni, 2011, Table A1）は、第1章から第6章で構築した熱弾性理論の実証的基盤である。このデータベースは、地球マントル全域の圧力・温度範囲をカバーする **46 種のエンドメンバー鉱物** の熱弾性パラメータを収録しており、地殻の長石から D'' 層の post-perovskite 相まで、マントルを構成する主要鉱物相を網羅している。

各エンドメンバーは、基準条件（$T_0 = 300$ K, $P = 0$ GPa）における一組の熱弾性パラメータによって特徴づけられる。具体的には、モル体積 $V_0$、等温体積弾性率 $K_0$ とその圧力微分 $K_0'$、剪断弾性率 $G_0$ とその圧力微分 $G_0'$、Debye 温度 $\theta_0$、Gruneisen パラメータ $\gamma_0$、その対数体積微分 $q_0$、剪断歪み Gruneisen パラメータ $\eta_{S0}$、化学式単位あたりの原子数 $n$、分子量、および基準 Helmholtz 自由エネルギー $F_0$ である。一部の相では、変位型相転移に対する Landau パラメータ（$T_{c0}$, $V_D$, $S_D$）や、鉄含有エンドメンバーに対する磁気パラメータ（スピン量子数 $S$、磁性原子数 $r$）が追加される。

データベースのパラメータ値は、広範な実験データ――静的圧縮（P-V-T）、熱量測定（比熱）、音速測定（Brillouin 散乱、超音波干渉法）、および相平衡境界――に対する**全体逆解析**（global inversion）によって決定された。この全体最適化により熱力学的自己無撞着性が保証される。すなわち、弾性的性質を再現するパラメータが同時に正しい相境界と熱的性質を予測する。

ThermoElasticCalculator における実装では、`SLB2011Endmembers` クラスがデータベースを `MineralParams` オブジェクトのリストとしてエンコードし、`SLB2011Solutions` クラスが固溶体モデル（相互作用パラメータ）を定義する。これにより、現実的なマントル鉱物学の計算が可能となる。

## 7.2 前提知識

本章を十分に理解するためには、以下の知識が必要である。

1. **Birch-Murnaghan 状態方程式と有限歪み理論**（第1章）：データベースの $K_0$ と $K_0'$ パラメータを用いた冷圧縮計算の基礎。Euler 有限歪み $f = [(V_0/V)^{2/3} - 1]/2$ の定義。

2. **Debye モデルと熱的性質**（第2章）：$\theta_0$ パラメータから任意の $P$-$T$ 条件における熱エネルギーとエントロピーを計算する Debye 関数の理論。

3. **Gruneisen パラメータと非調和性**（第3章）：$\gamma_0$, $q_0$, $\eta_{S0}$ の物理的意味。Gruneisen パラメータの体積依存性 $\gamma = \gamma_0 (V/V_0)^{q_0}$ の導出と応用。

4. **弾性率と地震波速度**（第4章）：$G_0$, $G_0'$, $\eta_{S0}$ を用いた高圧高温条件における剪断弾性率と地震波速度の計算手法。

5. **マントル鉱物の結晶化学**：かんらん石、ガーネット、輝石、ペロブスカイト構造の基礎的理解。イオン半径、配位数、結晶構造の対称性。

6. **固溶体熱力学**：理想混合と過剰混合、エンドメンバーの概念。$n$ 成分系における自由エネルギー最小化。

7. **Landau 理論**：石英の $\alpha$-$\beta$ 転移やスティショバイトの post-stishovite 転移など、変位型相転移の熱力学的記述。

8. **鉄含有鉱物の磁気寄与**：不対 $d$ 電子による常磁性エントロピーの基礎。

## 7.3 理論と方程式

### 7.3.1 エンドメンバーの全 Helmholtz 自由エネルギー

SLB2011 の中核となる方程式は、各エンドメンバーの全 Helmholtz 自由エネルギーの分解である：

$$F(V,T) = F_0 + F_{\text{cold}}(V) + F_{\text{Debye}}(V,T) + F_{\text{Landau}}(V,T) + F_{\text{mag}}(T)$$

各項の意味は以下の通りである：

- $F_0$: 基準条件 $(V_0, T_0)$ における参照 Helmholtz 自由エネルギー [kJ/mol]。元素の標準状態に対する相対値。
- $F_{\text{cold}}(V)$: 第3次 Birch-Murnaghan 状態方程式による冷圧縮歪みエネルギー。
- $F_{\text{Debye}}(V,T)$: Debye モデルによる格子振動の熱エネルギー。
- $F_{\text{Landau}}(V,T)$: Landau 変位型転移寄与（該当する相のみ）。
- $F_{\text{mag}}(T)$: 不対電子による磁気寄与（鉄含有相のみ）。

この分解は本質的である。各項が異なるデータベースパラメータの部分集合によって制御されるため、異なる種類の実験データがそれぞれ異なるパラメータを制約する。

### 7.3.2 冷圧縮エネルギー（第3次 Birch-Murnaghan）

冷圧縮エネルギーは Euler 有限歪み $f$ の関数として表される：

$$F_{\text{cold}} = \frac{9 K_0 V_0}{2} f^2 \left(1 + a_1 f\right)$$

ここで

$$a_1 = \frac{3(K_0' - 4)}{2}$$

であり、$f$ は Euler 有限歪みである：

$$f = \frac{1}{2}\left[\left(\frac{V_0}{V}\right)^{2/3} - 1\right]$$

この項はデータベースの 2 つのパラメータ $K_0$ と $K_0'$ のみに依存する。これらは静的圧縮実験によって最も良く制約されるパラメータである。

対応する Birch-Murnaghan 圧力は：

$$P(f) = 3 K_0 (1+2f)^{5/2} f \left[1 + \frac{3(K_0'-4)}{2} f\right]$$

ThermoElasticCalculator では `MineralParams.GetPressure(f)` メソッドがこの式を直接実装している。

### 7.3.3 有限歪みにおける等温体積弾性率

有限歪み $f$ における等温体積弾性率の冷圧縮項は：

$$K_{T,\text{cold}}(f) = K_0 (1 + 2f)^{5/2} \left[1 + (3K_0' - 5)f + \frac{27}{2}(K_0' - 4)f^2\right]$$

この式は `MineralParams.BM3KT(f)` メソッドで計算される。データベースの $K_0$ と $K_0'$ がこの関数を完全に決定する。

### 7.3.4 体積依存 Debye 温度と Gruneisen パラメータ

Debye 温度の体積依存性は、Gruneisen パラメータ $\gamma$ を通じて記述される：

$$\gamma(V) = \gamma_0 \left(\frac{V}{V_0}\right)^{q_0}$$

$$\theta(V) = \theta_0 \exp\left(\frac{\gamma_0 - \gamma}{q_0}\right)$$

3 つのデータベースパラメータ $\theta_0$, $\gamma_0$, $q_0$ がこの熱的寄与を制御する。圧縮下（$V < V_0$）では $\gamma < \gamma_0$ となるため Debye 温度は上昇し、格子は硬くなる。パラメータ $q_0$ は $\gamma$ の体積変化速度を制御する。

**物理的解釈**：$\gamma_0 \approx 1$ の鉱物（マントルにおける典型値）では、体積が 10% 減少すると Debye 温度は約 10% 上昇する。$q_0 > 1$ の場合、この効果は圧縮が進むにつれて弱まる（$\gamma$ がより速く減少するため）。$q_0 < 1$ の場合、効果は強まる。

### 7.3.5 剪断弾性率の熱補正

剪断弾性率は冷圧縮項と熱補正項の和である：

$$G(f,T) = G_{\text{cold}}(f) - \frac{\eta_{S0}}{V} \Delta E_{\text{th}}$$

ここで $G_{\text{cold}}(f)$ は有限歪みにおける剪断弾性率（BM3GT）であり、$G_0$, $G_0'$, $K_0$, $K_0'$ に依存する。$\eta_{S0}$ は剪断歪み Gruneisen パラメータであり、$\Delta E_{\text{th}}$ は基準温度からの熱エネルギー変化である。

有限歪みにおける冷圧縮剪断弾性率は：

$$G_{\text{cold}}(f) = G_0 (1 + 2f)^{5/2} \left[1 + A_s f + \frac{3K_0 A_s - 5G_0}{G_0} \cdot \frac{f^2}{2}\right]$$

ここで導出パラメータ $A_s$ はデータベースパラメータから計算される：

$$A_s = -2\gamma_0 - 2\eta_{S0}$$

ThermoElasticCalculator では `MineralParams.As` プロパティとしてこれを実装している。

### 7.3.6 磁気エントロピー寄与

鉄含有エンドメンバーの高温常磁性エントロピーは：

$$S_{\text{mag}} = n_r R \ln(2S + 1)$$

- $S$: 磁性イオンのスピン量子数（例：高スピン Fe$^{2+}$ では $S = 2$）
- $n_r$: 化学式単位あたりの磁性原子数
- $R$: 気体定数 $= 8.314$ J/(mol$\cdot$K)

マントル温度（Neel 温度・Curie 温度を十分に上回る）では、すべてのスピン微小状態が等しく占有されるため、完全な常磁性エントロピー $R \ln(2S+1)$ が実現する。高スピン Fe$^{2+}$（$S = 2$）の場合、$2S+1 = 5$ 個のスピン微小状態があり、1 磁性原子あたり $R \ln 5 = 13.38$ J/(mol$\cdot$K) のエントロピーに寄与する。

対応する自由エネルギー寄与は：

$$F_{\text{mag}} = -T \cdot S_{\text{mag}} = -n_r R T \ln(2S+1)$$

### 7.3.7 Landau 変位型転移寄与

変位型相転移（例：石英の $\alpha$-$\beta$ 転移）は Landau 理論によってモデル化される：

$$F_{\text{Landau}} = \frac{S_D}{2} T_c \left(1 - \frac{T}{T_c}\right)^2 \quad (T < T_c)$$

$$F_{\text{Landau}} = 0 \quad (T \geq T_c)$$

3 つの Landau パラメータが必要である：

- $T_{c0}$: 圧力ゼロにおける臨界温度 [K]
- $V_D$: 最大過剰体積 [cm$^3$/mol]
- $S_D$: 最大過剰エントロピー [J/(mol$\cdot$K)]

圧力下では臨界温度が変化する：

$$T_c(P) = T_{c0} + P \frac{V_D}{S_D}$$

SLB2011 の 46 エンドメンバーのうち、Landau パラメータを必要とするのは**石英**（$T_{c0} = 847$ K）と**スティショバイト**（$T_{c0} = -4250$ K）の 2 相のみである。

### 7.3.8 導出係数

データベースパラメータから、Mie-Gruneisen-Debye 形式に入る導出係数が計算される：

$$A_{ii} = 6\gamma_0$$

$$A_{iikk} = -12\gamma_0 + 36\gamma_0^2 - 18\gamma_0 q_0$$

$$A_s = -2\gamma_0 - 2\eta_{S0}$$

これらは `MineralParams` クラスのプロパティとして実装されている：

- `Aii` $= 6 \gamma_0$
- `Aiikk` $= -12\gamma_0 + 36\gamma_0^2 - 18\gamma_0 q_0$
- `As` $= -2\gamma_0 - 2\eta_{S0}$

これらの係数は、有限歪み展開における体積弾性率と剪断弾性率の熱補正に使用される。

## 7.4 物理概念

### 7.4.1 エンドメンバー鉱物

**定義**：固定化学組成を持つ鉱物相であり、固溶体系列の熱力学的構成要素（building block）として機能する。エンドメンバーは固溶体の純粋な組成端を表す。

**物理的直観**：あらゆる色が原色から混合できるように、あらゆるマントル岩石組成はエンドメンバー鉱物の混合として記述できる。Forsterite（$\mathrm{Mg_2SiO_4}$）と Fayalite（$\mathrm{Fe_2SiO_4}$）はかんらん石固溶体のエンドメンバーである。固溶体の熱力学的性質は、エンドメンバーの性質に混合項を加えることで構築される。

**よくある誤解**：

- エンドメンバーは必ずしも自然界で安定な相ではない。純粋な fayalite はマントルでは稀であるが、そのエンドメンバーパラメータは Fe 含有かんらん石の性質計算に不可欠である。
- 46 のエンドメンバーが 1 つの岩石中に 46 個の独立相として存在するわけではない。与えられた $P$-$T$ 条件において典型的なマントル鉱物集合体は 4--6 個の安定相のみを含み、各相がいくつかのエンドメンバーの固溶体である。

### 7.4.2 全体逆解析によるパラメータ決定

**定義**：すべてのエンドメンバーパラメータを、複数種類の実験データ（P-V-T、$C_p$、音速、相境界）に対して同時に最小二乗フィッティングし、熱力学的自己無撞着性を保証する手法。

**物理的直観**：実験の種類ごとに異なるパラメータの組み合わせが制約される。

| 実験手法 | 制約されるパラメータ |
|:---------|:-------------------|
| 静的圧縮 (P-V-T) | $V_0$, $K_0$, $K_0'$ |
| 熱量測定 ($C_p$) | $\theta_0$, $\gamma_0$ |
| 音速測定 (Brillouin 散乱) | $G_0$, $G_0'$, $\eta_{S0}$ |
| 相平衡境界 | $F_0$（相対自由エネルギー） |

すべてを同時にフィッティングすることで、パラメータ間のトレードオフが解消され、自己無撞着性が強制される。

**よくある誤解**：

- パラメータは各鉱物について独立に測定され組み立てられるのではない。全体逆解析では、1 つのパラメータ（例：forsterite の $K_0$）を変更すると、他のパラメータ（例：forsterite の $\gamma_0$ や wadsleyite の $F_0$）もフィットを維持するために移動する。
- パラメータの不確実性は相関している。$K_0 = 128 \pm 2$ GPa と報告するとき、$K_0'$ との相関を無視するのは誤解を招く。

### 7.4.3 鉱物ファミリーと構造的類似性

**定義**：同じ結晶構造を共有する鉱物群は 1 つのファミリーを形成する（例：かんらん石ファミリー = forsterite + fayalite）。ファミリー内では特定のパラメータが等しいか、系統的な関係にあるように制約されることが多い。

**物理的直観**：Mg と Fe は類似したイオン半径と結合特性を持つため、結晶構造中で Mg を Fe に置換すると質量と結合強度は変化するが、構造的トポロジーは保存される。このため、圧力微分（$K_0'$, $G_0'$）や体積指数 $q_0$ はファミリー内で等しいと仮定されることが多い一方、$V_0$, $K_0$, $G_0$, $\theta_0$ は組成に伴い系統的に変化する。

**重要な区別**：かんらん石-wadsleyite-ringwoodite の変態系列は同じ化学組成（$\mathrm{Mg_2SiO_4}$）を持つが、異なる結晶構造であるため、それぞれ独立のパラメータ集合を持つ別のファミリーである。

### 7.4.4 基準状態の規約

**定義**：SLB2011 のすべてのパラメータは $T_0 = 300$ K, $P = 0$ GPa を基準とする。Helmholtz 自由エネルギー $F_0$ は元素の標準状態に対する相対値として定義される。

**物理的直観**：すべてのエンドメンバーに共通の基準状態を選ぶことで、自由エネルギーの直接比較が可能となり、任意の $P$-$T$ 条件で熱力学的に安定な相を決定できる。基準温度を 300 K（0 K ではなく）に選ぶのは、零点エネルギーの複雑さを避け、室温の実験室測定との接続を簡素化するためである。

**よくある誤解**：

- $F_0$ は鉱物の全エネルギーではなく、元素の基準に依存する相対量である。同じ組成の相間の $F_0$ の差のみが相平衡計算において物理的意味を持つ。
- $V_0$ は 300 K, 1 bar における体積であり、0 K における体積ではない。0 K から 300 K までの熱膨張は基準体積にすでに含まれている。

### 7.4.5 Landau 変位型転移パラメータ

**定義**：3 つのパラメータ（$T_{c0}$, $V_D$, $S_D$）が、結合を切断せずに原子位置がシフトする二次（または弱い一次）構造相転移を記述する。

**物理的直観**：石英において、$\alpha$-$\beta$ 転移（1 bar で 847 K）は $\mathrm{SiO_4}$ 四面体のわずかな回転を伴う。$T_c$ 以下では構造が歪み（対称性が低く、体積が小さい）、Landau パラメータはこの歪みに伴う過剰体積 $V_D$ と過剰エントロピー $S_D$ を捉える。

**重要な注意**：スティショバイトの $T_{c0} = -4250$ K は負であるが、これは負の温度で転移が起こることを意味しない。高圧下で $T_c(P) = T_{c0} + P \cdot V_D/S_D$ が正となるとき初めて転移が実現する。Landau パラメータは、かんらん石から wadsleyite への再構成型転移とは本質的に異なる概念であり、再構成型転移は別個のエンドメンバーの自由エネルギー比較によって扱われる。

### 7.4.6 鉄含有相の磁気寄与

**定義**：鉄含有エンドメンバーは不対 $d$ 電子を持ち、高温で常磁性エントロピーに寄与する。これはスピン量子数 $S$ と磁性原子数 $r$ によってパラメータ化される。

**物理的直観**：高スピン Fe$^{2+}$ は 4 個の不対電子（$S = 2$）を持ち、$2S+1 = 5$ 個のスピン微小状態がある。磁気秩序温度（ケイ酸塩では通常 100 K 以下）を十分に上回るマントル温度では、すべての微小状態が等確率で占有され、Fe 原子 1 個あたり $R \ln 5$ のエントロピーに寄与する。これは高温において鉄含有相を鉄不含有相に対して安定化させる。

**よくある誤解**：

- SLB2011 の磁気寄与は高温常磁性極限である。低温の磁気秩序（反強磁性 Neel 転移）のモデル化は行われておらず、マントル温度では無関係である。
- すべての鉄含有鉱物が同じ磁性原子数 $r$ を持つわけではない。Fe-wadsleyite では $r = 8$（選択された化学式基準における Fe サイト数を反映）であり、fayalite では $r = 2$ である。

## 7.5 計算例

### 7.5.1 ブリッジマナイト（ペロブスカイト）エンドメンバーの比較

**問題**：3 種のブリッジマナイトエンドメンバー（Mg, Fe, Al）のパラメータ集合を比較し、共有パラメータと異なるパラメータを特定せよ。この構造がパラメータ制約について何を意味するか考察せよ。

**解法**：`SLB2011Endmembers.GetAll()` から mpv, fpv, apv のパラメータを抽出する。

**パラメータの比較**：

| パラメータ | Mg-Pv (mpv) | Fe-Pv (fpv) | Al-Pv (apv) | 共有? |
|:----------|:------------|:------------|:------------|:------|
| $n$ (原子数) | 5 | 5 | 5 | Yes |
| $K_0'$ | 4.14 | 4.14 | 4.14 | Yes |
| $\gamma_0$ | 1.56508 | 1.56508 | 1.56508 | Yes |
| $q_0$ | 1.10945 | 1.10945 | 1.10945 | Yes |
| $V_0$ [cm$^3$/mol] | 24.445 | 25.485 | 24.944 | No |
| $K_0$ [GPa] | 250.53 | 272.12 | 258.20 | No |
| $G_0$ [GPa] | 172.90 | 132.68 | 171.31 | No |
| $G_0'$ | 1.6904 | 1.3748 | 1.4971 | No |
| $\theta_0$ [K] | 905.94 | 870.81 | 886.46 | No |
| $\eta_{S0}$ | 2.5654 | 2.2921 | 2.4713 | No |
| $F_0$ [kJ/mol] | $-1368.283$ | $-1040.920$ | $-1533.878$ | No |
| $S$ (スピン) | -- | 2.0 | -- | -- |
| $r$ (磁性原子数) | -- | 1.0 | -- | -- |

**考察**：

共有されている $K_0'$, $\gamma_0$, $q_0$ は結晶構造のトポロジーに関連するパラメータであり、組成に依存しないと仮定されている。一方、$V_0$, $K_0$, $G_0$, $\theta_0$ は化学結合の強さと質量に依存するため、組成ごとに異なる。

Fe 置換は $V_0$ と $K_0$ を増加させる（より重い原子、わずかに圧縮されやすい結合）が、$G_0$ を著しく減少させる（Fe-O 結合は剪断に対して弱い）。Fe-ブリッジマナイトの $F_0$ が大幅に小さい（より正に近い）のは、ブリッジマナイト構造における鉄の熱力学的安定性が相対的に低いことを反映する。Al-ブリッジマナイトが最も負の $F_0$ を持ち、Al がブリッジマナイト構造を安定化させることと整合する。

### 7.5.2 鉄含有鉱物の磁気エントロピー計算

**問題**：Fayalite（$\mathrm{Fe_2SiO_4}$）と Fe-wadsleyite のマントル温度における磁気エントロピー寄与を計算し、相安定性への影響を評価せよ。

**解法**：

$$S_{\text{mag}} = n_r R \ln(2S+1)$$

**Fayalite**（$S = 2$, $r = 2$）：

$$S_{\text{mag}} = 2 \times 8.314 \times \ln(2 \times 2 + 1) = 2 \times 8.314 \times \ln 5$$

$$= 2 \times 8.314 \times 1.6094 = 26.76 \;\text{J/(mol·K)}$$

**Fe-Wadsleyite**（$S = 2$, $r = 8$）：

$$S_{\text{mag}} = 8 \times 8.314 \times \ln 5 = 8 \times 8.314 \times 1.6094 = 107.04 \;\text{J/(mol·K)}$$

**マントル温度 $T = 1500$ K における磁気自由エネルギー寄与**：

$$F_{\text{mag}} = -T \cdot S_{\text{mag}}$$

Fayalite: $F_{\text{mag}} = -1500 \times 26.76 \times 10^{-3} = -40.14$ kJ/mol

Fe-Wadsleyite: $F_{\text{mag}} = -1500 \times 107.04 \times 10^{-3} = -160.56$ kJ/mol

**考察**：

Fayalite と Fe-wadsleyite の磁性原子数の大きな違い（$r = 2$ vs $r = 8$）は、データベースにおける化学式単位の規約の違いを反映している。Fe-wadsleyite は $\mathrm{Fe_2SiO_4}$ の 4 化学式分に相当する基準を使用しており（$n = 7$ 原子は 1 化学式分に対応するが、$r = 8$ の磁性原子が分配される）、この規約を注意深く追跡しないと計算誤差が生じる。$T \cdot S_{\text{mag}}$ 項は大きく、鉄含有多形間の相境界に顕著な影響を及ぼす。

### 7.5.3 石英の異常な Gruneisen パラメータ

**問題**：石英のほぼゼロの Gruneisen パラメータ（$\gamma_0 = -0.00296$）の物理的意味とその熱的性質への帰結を説明せよ。

**解法**：

Gruneisen パラメータは熱圧力と熱エネルギーの関係を表す：

$$P_{\text{th}} = \frac{\gamma E_{\text{th}}}{V}$$

$\gamma \approx 0$ は、熱圧力が無視できることを意味する。

**石英のパラメータ**：

- $\gamma_0 = -0.00296$（ほぼゼロ、わずかに負）
- $\theta_0 = 816.33$ K
- $K_0 = 49.55$ GPa
- $K_0' = 4.33$
- Landau: $T_{c0} = 847$ K, $V_D = 1.222$ cm$^3$/mol, $S_D = 5.164$ J/(mol$\cdot$K)

体積膨張率 $\alpha$ は $\gamma C_v / (K_T V)$ に比例するため、$\gamma \approx 0$ は異常に小さい熱膨張を意味する。

**物理的解釈**：

石英のほぼゼロ（わずかに負の）Gruneisen パラメータは、$\mathrm{SiO_2}$ の開いた骨格構造に起因する。密充填酸化物と比較して、石英は非常に開いた結晶構造を持ち、熱振動は顕著な体積変化なしに収容できる。これは物理的に合理的である。

実際には、$\alpha$-$\beta$ 転移温度（847 K）付近の石英の熱的挙動は、Debye モデルではなく Landau パラメータによって支配される。Landau 転移は過剰体積 $V_D = 1.222$ cm$^3$/mol と過剰エントロピー $S_D = 5.164$ J/(mol$\cdot$K) を通じて支配的な熱効果を提供し、熱膨張を Debye モデルから事実上分離している。

**計算メソッド**：

| メソッド | 機能 |
|:--------|:-----|
| `GetPressure(f)` | 有限歪み $f$ における BM3 圧力の計算 |

| `BM3GT(f)` | 有限歪み $f$ における剪断弾性率の冷圧縮項 |

### 7.6.3 BM3Finite の求根アルゴリズム

`BM3Finite(P)` は Birch-Murnaghan 状態方程式の逆問題を解く。すなわち、目標圧力 $P$ が与えられたとき、$P(f) = P_{\text{target}}$ を満たす有限歪み $f$ を求める。

アルゴリズムの概要：

1. **初期推定**：$f_{\text{est}} = P/(3K_0)$ として線形近似
2. **上限設定**：安全マージンを加えた動的上界を設定
3. **Regula Falsi 法**による反復求根
4. **収束判定**：計算機精度まで収束

46 エンドメンバーについて、マントルの任意の圧力（最大 140 GPa）において典型的に 30 回以下の反復で収束する。

### 7.6.4 固溶体データベース

`SLB2011Solutions` クラスは Table A2 の相互作用パラメータを実装する。各 `InteractionEntry` は固溶体名、2 つのエンドメンバーの略称、および Margules パラメータ $W$ [kJ/mol] を含む。

主要な固溶体とその相互作用パラメータの例：

| 固溶体 | エンドメンバー対 | $W$ [kJ/mol] |
|:------|:---------------|:------------|
| Olivine | fo-fa | 7.6 |
| Wadsleyite | mw-fw | 16.5 |
| Ringwoodite | mrw-frw | 9.1 |
| Ferropericlase | pe-wu | 13.0 |
| Clinopyroxene | di-he | 24.0 |
| Garnet | py-gr | 21.0 |
| Perovskite | mpv-fpv | 0.0（理想混合） |

$W = 0$ は理想混合を意味し、エンドメンバー間の相互作用がないことを示す。ブリッジマナイトとポストペロブスカイトの全エンドメンバー対で $W = 0$ となっており、これは実験データの不足を反映している可能性がある。

### 7.6.5 データの入出力

`MineralParams` クラスは JSON および CSV 形式のインポート/エクスポートをサポートする：

- `ExportJson()` / `ImportJson()`: JSON 形式での入出力
- `ExportCsvRow()` / `ImportCsvRow()`: CSV 行形式での入出力

これにより、パラメータの探索や感度解析が外部ツールとの連携で容易に行える。

### 7.7.3 導出係数の検証

コード中の導出係数を手計算で検証する。Forsterite（$\gamma_0 = 0.99282$, $q_0 = 2.10672$, $\eta_{S0} = 2.2997$）の場合：

$$A_{ii} = 6 \times 0.99282 = 5.95692$$

$$A_{iikk} = -12 \times 0.99282 + 36 \times 0.99282^2 - 18 \times 0.99282 \times 2.10672$$

$$= -11.9138 + 35.4843 - 37.6489 = -14.0784$$

$$A_s = -2 \times 0.99282 - 2 \times 2.2997 = -1.9856 - 4.5994 = -6.5850$$

これらの値は `MineralParams` クラスのプロパティ `Aii`, `Aiikk`, `As` と一致するはずである。

### 7.8.2 パラメータの系統的傾向

46 エンドメンバーのパラメータには明確な系統的傾向が存在する。

**体積弾性率 $K_0$ とモル体積 $V_0$ の関係（Birch の法則）**：

高圧相（ブリッジマナイト、ポストペロブスカイト、スティショバイト）は低い $V_0$ と高い $K_0$ のクラスターを形成し、低圧相（石英、アルバイト、ネフェリン）は高い $V_0$ と低い $K_0$ のクラスターを形成する。この傾向は経験的 Birch の法則 $K_0 V_0^{4/3} \approx$ 一定 と概ね整合する。

代表的な値：

| 鉱物 | $V_0$ [cm$^3$/mol] | $K_0$ [GPa] | $V_0 \cdot K_0$ |
|:-----|:-------------------|:------------|:----------------|
| Quartz | 23.670 | 49.55 | 1173 |
| Albite | 100.452 | 59.76 | 6003 |
| Forsterite | 43.603 | 127.96 | 5579 |
| Mg-Pv | 24.445 | 250.53 | 6125 |
| Stishovite | 14.017 | 314.34 | 4407 |
| Seifertite | 13.670 | 327.58 | 4476 |

**Debye 温度 $\theta_0$ の系統性**：

ルチル型構造の $\mathrm{SiO_2}$ 多形（stishovite: $\theta_0 = 1108$ K, seifertite: $\theta_0 = 1141$ K）は最も高い Debye 温度を持ち、硬い Si-O 結合を反映する。一方、wuestite（$\theta_0 = 454$ K）は最も低い値の一つであり、より弱い Fe-O 結合と整合する。

**$K_0'$ の値域**：

多くのエンドメンバーで $K_0' \approx 4$--5 であるが、直方輝石系（enstatite, ferrosilite: $K_0' = 7.03$）は著しく高い値を示す。有限歪み理論は無限圧力極限で $K_0' \to 4$ を予測するため、$K_0'$ の大きな偏差は結晶構造の異方性や特異な圧縮挙動を反映する。

**制約の程度の違い**：

パラメータの有効桁数はデータの制約度合いを反映する。例えば：

- Forsterite: $K_0 = 127.96$ GPa（5桁 → よく制約されている）
- Na-Majorite: $K_0 = 163.0$ GPa, $K_0' = 5.0$, $G_0' = 1.7$（丸い数値 → 推定値の可能性）
- CaFerrite 型: $K_0 = 212.0$ GPa, $K_0' = 4.1$（丸い数値 → 限られた実験データ）

## 7.9 歴史的背景

### 7.9.1 鉱物データベースの発展

SLB2011 データベースは、マントル鉱物の自己無撞着な熱力学的枠組みを構築する数十年の努力の集大成である。

**1970-80 年代**：初期の鉱物データベース（例：Berman 1988, Holland & Powell 1990）は地殻・上部マントルの鉱物相に焦点を当て、物理に基づく状態方程式ではなく経験的な熱膨張・圧縮率関数を使用していた。

**1990 年**：Stixrude & Bukowinski (1990) が、Birch-Murnaghan 状態方程式と Mie-Gruneisen-Debye 形式をマントル鉱物に初めて適用し、微視的な振動特性（Debye 温度、Gruneisen パラメータ）を巨視的な観測量に結びつけた。

**2005 年（Part I）**：Stixrude & Lithgow-Bertelloni (2005) が理論的枠組みを体系化。有限歪み弾性論、剪断弾性率の取り扱い、Mie-Gruneisen-Debye 熱モデルの統合的な定式化を提示した。

**2011 年（Part II）**：同著者らが 46 エンドメンバーのパラメータを全体逆解析によって決定し、Table A1 として発表。これがSLB2011 データベースの中核であり、ThermoElasticCalculator の `SLB2011Endmembers` クラスが直接実装するものである。

**2012 年以降**：データベースは地球物理学コミュニティに急速に採用され、BurnMan（Cottaar et al., 2014）や HeFESTo などのツールに統合された。Pyrolite、harzburgite、MORB 組成に対する地震波速度プロファイルの計算に広く使用されている。

### 7.9.2 競合するデータベース

**Holland-Powell データベース（HP11）**：異なる熱圧力定式化を使用し、定期的に更新される競合データベース。SLB2011 との比較はパラメータ化の選択に伴う不確実性を浮き彫りにする。HP11 は Mie-Gruneisen 熱圧力アプローチを採用するが、状態方程式の定式化が異なる。

**第一原理データベース**：実験フィッティングではなく密度汎関数理論からパラメータを導出する新しいアプローチ。実験データが乏しい相に対して独立の制約を提供できるが、理論的不確実性（交換相関汎関数の選択など）が残る。

### 7.9.3 現在の発展

2020 年代において、SLB 枠組みの拡張が進んでいる：

- **含水鉱物相**の追加（phase D, superhydrous phase B, $\delta$-AlOOH）
- **スピンクロスオーバー**の取り込み（フェロペリクレース、ブリッジマナイト中の Fe のスピン転移）
- **非弾性効果**の実装（ThermoElasticCalculator の `AnelasticityDatabase` に対応）

しかし、核心となる 46 エンドメンバー無水データベースは依然として標準的参照として広く使用されている。

## 7.10 未解決課題

### 7.10.1 鉄含有鉱物のスピンクロスオーバー

SLB2011 は高スピンパラメータのみを使用するが、フェロペリクレースやブリッジマナイト中の Fe$^{2+}$ は 40--60 GPa 付近で高スピンから低スピンへの転移を起こす。この転移は $V_0$, $K_0$ などの物性を著しく変化させるが、データベースへの系統的な取り込み方法は確立されていない。

### 7.10.2 CaFerrite 型相のパラメータ制約

Mg-CaFerrite, Fe-CaFerrite, Na-CaFerrite のパラメータは丸い数値（$K_0' = 4.1$, $G_0' = 1.7$ など）を使用しており、限られた実験的制約を反映する。NAL（New Aluminous Phase）とも呼ばれるこれらの相は、下部マントルにおける Al と Na の担い手として重要であるが、その弾性パラメータの精度は他の相に比べて低い。

### 7.10.3 準調和近似を超える非調和効果

SLB2011 の Debye モデルは準調和近似に基づくが、最高マントル温度（$> 3000$ K）では固有非調和性が無視できなくなる可能性がある。これは有効パラメータ値を温度依存的にシフトさせ得る。

### 7.10.4 含水相の不在

現在の 46 エンドメンバーは無水マントルを仮定する。沈み込み帯における水の輸送をモデル化するには、phase D, superhydrous phase B, $\delta$-AlOOH などの含水鉱物相の追加が必要である。

### 7.10.5 相関するパラメータ不確実性の伝播

全体逆解析からの正式な共分散行列が公表されていないため、パラメータ不確実性の厳密な伝播が困難である。計算された地震波速度が相関するパラメータ不確実性にどれほど敏感であるかの定量的評価は未解決の課題である。

### 7.10.6 Na-Majorite パラメータの妥当性

Na-majorite のパラメータは丸い数値（$K_0' = 5.0$, $G_0' = 1.7$, $q_0 = 1.5$）を示しており、よく制約されたフィット結果というよりも推定値である可能性がある。遷移帯における Na 含有ガーネットの正確な地震波速度予測のためには、これらのパラメータの改善が望まれる。

### 7.10.7 2011 年以降の実験データの反映

2011 年以降、特にブリッジマナイトとポストペロブスカイトに対するメガバール圧力での新しい Brillouin 散乱測定が報告されている。これらの新データをどのようにデータベースに組み込むべきかは、全体逆解析の再実行を要するため容易ではない。

## 7.11 演習問題

### 演習 7.1: エンドメンバーの分類と共有パラメータ

46 エンドメンバーすべてを鉱物ファミリーごとに分類せよ。各ファミリーについて、エンドメンバー間で共有（等しいと制約）されているパラメータと、異なるパラメータを特定せよ。共有される制約を正当化する物理的理由を議論せよ。

**ヒント**：ブリッジマナイト系では $K_0' = 4.14$, $\gamma_0 = 1.56508$, $q_0 = 1.10945$ が 3 つのエンドメンバーで共有されている。他のファミリーでも同様のパターンを探せ。

### 演習 7.2: ゼロ圧力密度の計算

SLB2011 データベースの $V_0$ と分子量から各エンドメンバーのゼロ圧力密度を計算せよ：

$$\rho_0 = \frac{M}{V_0}$$

ここで $M$ は分子量 [g/mol]、$V_0$ はモル体積 [cm$^3$/mol] である。密度の低い順にランク付けし、結晶構造（配位数、充填効率）との関係を論じよ。

**数値確認**：Forsterite の場合、$\rho_0 = 140.69 / 43.603 = 3.227$ g/cm$^3$。

### 演習 7.3: かんらん石-wadsleyite 速度コントラスト

Forsterite と Mg-Wadsleyite について、$P = 10$ GPa, $T = 1500$ K における $K_T$, $G$, $V_p$, $V_s$, $\rho$ を完全な Mie-Gruneisen-Debye 計算により求めよ。両者の速度コントラストを計算し、410 km 不連続面の地震学的観測と比較せよ。

**手順の概要**：
1. `BM3Finite(P)` で $f$ を求める
2. $V = V_0 / (1+2f)^{3/2}$ でモル体積を求める
3. $\gamma(V)$, $\theta(V)$ を計算する
4. Debye 関数から $E_{\text{th}}$, $C_v$ を計算する
5. $K_T$, $K_S$, $G$ の熱補正を適用する
6. $V_p = \sqrt{(K_S + 4G/3)/\rho}$, $V_s = \sqrt{G/\rho}$ を計算する

### 演習 7.4: Landau パラメータの効果

石英の体積とエントロピーを 300 K から 1200 K まで、Landau 寄与ありとなしで温度の関数としてプロットせよ。$\alpha$-$\beta$ 転移はどこで発生するか？Landau 項がない場合の振る舞いと比較して議論せよ。

### 演習 7.5: 全鉄含有エンドメンバーの磁気エントロピー

すべての鉄含有エンドメンバーについて磁気エントロピーを計算せよ。1 mol あたりの磁気エントロピーが最大の相はどれか？Fe 原子 1 個あたりでは？この結果が共存相間の Fe 分配係数にどう影響するか論じよ。

### 演習 7.6: 体積弾性率の系統性

全鉱物ファミリーにわたって $K_0$ と $K_0'$ を比較せよ。高圧相（ブリッジマナイト、ポストペロブスカイト）は低圧相（かんらん石、輝石）より系統的に高い $K_0$ と低い $K_0'$ を持つか？有限歪み理論が予測する無限圧力極限 $K_0' \to 4$ との関連を論じよ。

### 演習 7.7: パラメータ感度解析

Mg-ブリッジマナイトの $K_0$ を $\pm 5$ GPa 変化させ、50 GPa, 2000 K における $V_p$ と $V_s$ を再計算せよ。速度変化は地震学的分解能（$\sim 0.1\%$）と比較してどの程度か？$G_0$ と $\theta_0$ についても同様の解析を繰り返せ。

### 演習 7.8: データのインポート/エクスポート

`MineralParams.ExportCsvFile()` を用いて全データベースを CSV にエクスポートし、1 つのパラメータ（例：periclase の $G_0$ を 10% 増加）を変更して再インポートせよ。変更前後の速度を比較し、パラメータ感度を実感せよ。

## 7.14 他章との関連

### 第1章との関連（前提）

第1章で導出した Birch-Murnaghan 状態方程式は、データベースの $K_0$ と $K_0'$ パラメータを直接使用する。SLB2011 のすべての冷圧縮計算は第1章の BM3 表現に基づく。特に、`MineralParams.GetPressure(f)` メソッドは第1章の式 (1.xx) を実装している。

### 第2章との関連（前提）

第2章で展開した Debye モデルは、$\theta_0$ パラメータを入力として使用する。Debye 関数の評価と比熱計算が $\theta_0$ を任意の $P$-$T$ 条件における熱エネルギーとエントロピーに変換する。

### 第3章との関連（前提）

第3章は $\gamma_0$, $q_0$, $\eta_{S0}$ の物理的意味を説明する。Gruneisen パラメータの体積依存性 $\gamma = \gamma_0(V/V_0)^{q_0}$ は、熱圧力と弾性的性質の補正の計算に中心的な役割を果たす。

### 第4章との関連（前提）

第4章は $G_0$, $G_0'$, $\eta_{S0}$ を用いて高圧高温条件における剪断弾性率と地震波速度を計算する方法を示す。BM3GT 有限歪み表現は $G_0$, $G_0'$, $K_0$, $K_0'$ を同時に使用する。

### 第8章（固溶体混合、予定）との関連（発展）

本章のエンドメンバーデータベースは固溶体モデルの基盤である。第8章では、46 エンドメンバーに `SLB2011Solutions` クラスの相互作用パラメータ（Margules パラメータ）を加えることで、ガーネットや輝石などの多成分相における非理想混合を記述する。

### 第9章（相平衡、予定）との関連（応用）

データベースの $F_0$ 値は競合する相の相対的安定性を決定する。Gibbs 自由エネルギー最小化にこれらの $F_0$ 値（$PV$ 項と $-TS$ 項を加えて）を使用することで、マントル地温勾配に沿った各 $P$-$T$ 条件での安定鉱物集合体を予測する。

## 7.15 補足：パラメータの物理的範囲と妥当性チェック

実装やデータの修正において、パラメータ値が物理的に妥当な範囲にあることを確認することは重要である。以下に各パラメータの典型的な範囲を示す：

| パラメータ | 物理的に妥当な範囲 | 根拠 |
|:---------|:----------------|:-----|
| $V_0$ | 10--170 cm$^3$/mol | 最小：periclase (11.2), 最大：spinel (159.0) |
| $K_0$ | 40--330 GPa | 最小：quartz (49.6), 最大：seifertite (327.6) |
| $K_0'$ | 3.5--7.5 | 理論的極限 $K_0' = 4$（無限圧力）付近。直方輝石は例外的に高い |
| $G_0$ | 30--230 GPa | 最小：nepheline (30.7), 最大：seifertite (227.5) |
| $G_0'$ | 0.3--2.3 | 最小：spinel/hercynite (0.37), 最大：periclase (2.14) |
| $\theta_0$ | 450--1150 K | 最小：wuestite (454), 最大：seifertite (1141) |
| $\gamma_0$ | $-0.01$--1.9 | 最小：quartz ($-0.003$), 最大：Mg-PPv (1.89) |
| $q_0$ | 0.2--3.5 | 最小：HP-clinopyroxene (0.20), 最大：enstatite (3.44) |
| $\eta_{S0}$ | $-0.06$--5.0 | 最小：wuestite ($-0.057$), 最大：seifertite (4.97) |

負の $\gamma_0$（石英）や負の $\eta_{S0}$（wuestite）は物理的に許容されるが、大多数のマントル鉱物では正の値をとる。パラメータ修正時にこれらの範囲から大きく外れる値が現れた場合は、入力データの誤りを疑うべきである。

## 7.16 まとめ

SLB2011 鉱物データベースは、マントル鉱物の熱弾性的性質を自己無撞着に記述する 46 エンドメンバーのパラメータ集合である。本章で学んだ主要な点を整理する：

1. **全 Helmholtz 自由エネルギーの分解**：冷圧縮項、Debye 熱項、Landau 項、磁気項の4つの寄与に分解され、各項がデータベースパラメータの異なる部分集合によって制御される。

2. **全体逆解析**：パラメータは個別の実験から独立に決定されるのではなく、P-V-T、比熱、音速、相境界のデータに対する同時フィッティングによって熱力学的自己無撞着性が保証される。

3. **鉱物ファミリーの構造**：同一結晶構造を共有するエンドメンバー群では、構造トポロジーに関連するパラメータ（$K_0'$, $\gamma_0$, $q_0$ など）が共有され、組成依存パラメータ（$V_0$, $K_0$, $G_0$, $\theta_0$ など）が異なる。

4. **特殊パラメータ**：Landau パラメータ（石英、スティショバイトの 2 相のみ）と磁気パラメータ（鉄含有エンドメンバー）は選択的に適用される。

5. **ThermoElasticCalculator との対応**：`SLB2011Endmembers`, `SLB2011Solutions`, `MineralDatabase`, `MineralParams` の各クラスが本章の理論を忠実に実装している。

6. **制約の不均一性**：forsterite や periclase のようによく制約されたエンドメンバーから、CaFerrite 型や Na-majorite のように限られたデータに基づくエンドメンバーまで、制約の程度に大きな差がある。

7. **未解決課題**：スピンクロスオーバー、含水相、非調和効果、および相関するパラメータ不確実性の取り扱いが主要な将来課題である。

本章のデータベースは、続く第8章（固溶体混合）および第9章（相平衡）の基盤であり、理論的枠組み（第1--4章）と地球物理学的応用を繋ぐ中核的役割を果たす。

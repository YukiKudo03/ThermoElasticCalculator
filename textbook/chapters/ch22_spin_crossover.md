# 第22章 フェロペリクレースにおける鉄スピンクロスオーバー

## 22.1 概要

鉄スピンクロスオーバーとは、八面体配位における $\text{Fe}^{2+}$ が、圧力の増加に伴い高スピン（HS: High-Spin）状態から低スピン（LS: Low-Spin）状態へと遷移する電子状態の変化である。HS 状態では 4 個の不対電子（$S=2$、スピン多重度 $g=5$）が存在するのに対し、LS 状態ではすべての電子が対をなし（$S=0$、$g=1$）不対電子は存在しない。

フェロペリクレース $(\text{Mg},\text{Fe})\text{O}$ はブリッジマナイトと共に下部マントルの主要鉄含有相であり、この鉱物中のスピンクロスオーバーはマントル温度において約 40--90 GPa（深さ約 1000--2100 km に対応）の広い圧力範囲にわたって生じる。この遷移の物理的起源は、結晶場分裂エネルギー（高圧で電子対形成を促進）と電子間反発エネルギー（低圧で不対電子を安定化）との競合にある。鋭い一次相転移とは異なり、有限温度でのスピンクロスオーバーは HS と LS の $\text{Fe}^{2+}$ イオンが格子上で配置エントロピーを伴って共存するため、連続的かつ熱的にブロードな過程となる。

与えられた圧力と温度における平衡低スピン分率 $n_{LS}$ は、HS-LS 混合状態の全自由エネルギーを最小化することで決定される。LS 端成分は HS 端成分と比較して約 $8\%$ の体積減少、約 $15\%$ の体積弾性率増加を示し、磁気エントロピーはゼロ（$S=0$ より $\ln(1)=0$）となる。

地震学的観点から、スピンクロスオーバーは混合スピン領域において体積弾性率 $K_S$ の異常な軟化を生じさせる。この軟化は圧縮波速度 $V_p$ やバルク音速を低下させ、中部下部マントルにおいて温度や組成の不均質と誤解されうる速度異常を生じさせる可能性がある。さらに、スピンクロスオーバーはフェロペリクレースとブリッジマナイト間の鉄分配にも影響を与え、共存相間の鉄の再分配を通じて下部マントルの集合的地震波物性をさらに修正する。

本コードベースの `SpinCrossoverCalculator` クラスはこの物理を実装する。HS 端成分（典型的には SLB2011 データベースのウスタイト 'wu'）を入力とし、`CreateLSEndmember` により近似的な LS 端成分を生成する。平衡 $n_{LS}$ は混合自由エネルギーに対する二段階グリッドサーチで求められ、有効的な $V$、$K_S$、$G_S$、$V_p$、$V_s$ は線形混合により計算される。

## 22.2 前提知識

本章を十分に理解するためには、以下の知識が必要である。

1. **熱力学ポテンシャル**（第1章）：Helmholtz 自由エネルギー $F(V,T)$ と Gibbs 自由エネルギー $G(P,T)$、および自由エネルギー最小化による平衡状態の決定。

2. **SLB2011 状態方程式の枠組み**（第2--6章）：Birch-Murnaghan 冷圧縮項、Mie-Gruneisen-Debye 熱モデル、$F = F_0 + F_{\text{cold}} + F_{\text{thermal}}$ の完全な分解。

3. **弾性率と地震波速度**（第4章）：体積弾性率 $K_S$、せん断弾性率 $G$、および地震波速度との関係：

$$V_p = \sqrt{\frac{K_S + \frac{4}{3}G}{\rho}}, \qquad V_s = \sqrt{\frac{G}{\rho}}$$

4. **結晶場理論**：八面体配位子場における 3d 軌道の $t_{2g}$ と $e_g$ への分裂、および分裂エネルギーと電子対形成エネルギーの競合による HS/LS 配置の決定。

5. **自由エネルギーへの磁気的寄与**（第9章）：磁気自由エネルギー $F_{\text{mag}} = -TrR\ln(2S+1)$ とスピン縮退による磁気エントロピーの概念。

6. **統計力学の基礎**：Boltzmann 分布、分配関数、二状態系の配置エントロピー。

7. **Mie-Gruneisen EOS ソルバー**（第6章）：`MieGruneisenEOSOptimizer` が指定された $P$, $T$ で平衡体積を求め、すべての熱力学量を計算する方法。

## 22.3 理論と方程式

### 22.3.1 HS-LS 混合系の自由エネルギー

スピンクロスオーバーの平衡状態は、HS 状態と LS 状態の混合自由エネルギーを最小化することで決定される。混合自由エネルギーは以下で与えられる：

$$F_{\text{mix}}(n_{LS}) = n_{LS} \, G_{LS} + (1 - n_{LS}) \, G_{HS} - T \, S_{\text{config}}(n_{LS})$$

ここで：

- $F_{\text{mix}}$：混合スピン状態系の全自由エネルギー [kJ/mol]
- $n_{LS}$：低スピン分率（0 から 1）
- $G_{HS}$：高スピン端成分の Gibbs 自由エネルギー [kJ/mol]
- $G_{LS}$：低スピン端成分の Gibbs 自由エネルギー [kJ/mol]
- $T$：温度 [K]
- $S_{\text{config}}$：スピン状態混合の配置エントロピー [J/mol/K]

低圧では $G_{HS} < G_{LS}$ であるため $n_{LS} = 0$（純粋 HS 状態）が安定であり、高圧では $G_{LS} < G_{HS}$ となり $n_{LS} = 1$（純粋 LS 状態）が安定化する。中間領域では配置エントロピーが混合状態を安定化する。コード中では `SpinCrossoverCalculator.ComputeFMix` に実装されており、$S_{\text{config}}$ の J/mol/K から kJ/mol/K への単位変換のために $1/1000$ の係数が適用される。

### 22.3.2 電子縮退を含む配置エントロピー

配置エントロピーは電子縮退因子を含む以下の式で与えられる：

$$S_{\text{config}} = -R \left[ n_{LS} \ln\frac{n_{LS}}{g_{LS}} + (1 - n_{LS}) \ln\frac{1 - n_{LS}}{g_{HS}} \right]$$

ここで：

- $R = 8.31477$ J/mol/K：気体定数
- $g_{LS} = 2 \times 0 + 1 = 1$：LS 状態の電子縮退度（$\text{Fe}^{2+}$ の場合）
- $g_{HS} = 2 \times 2 + 1 = 5$：HS 状態の電子縮退度（$\text{Fe}^{2+}$ の場合）

$g_{HS} = 5$ の項は高温において平衡を HS 状態へ強くバイアスさせる。HS イオンは LS イオンの 5 倍の縮退スピン配向を持つためである。$n_{LS} = 0$（純粋 HS）のとき、$S_{\text{config}} = R\ln(g_{HS}) = R\ln(5)$ となり、磁気エントロピーが回復される。境界処理（$n$ が 0 または 1 近傍）は `ComputeFMix` 内で $\log(0)$ を回避するよう明示的に実装されている。

### 22.3.3 スピンクロスオーバーの平衡条件

$F_{\text{mix}}$ の $n_{LS}$ に関する偏微分をゼロとおくと、Boltzmann 型の平衡条件が得られる：

$$\frac{\partial F_{\text{mix}}}{\partial n_{LS}} = 0 \implies G_{LS} - G_{HS} + RT \ln\frac{n_{LS} \, g_{HS}}{(1 - n_{LS}) \, g_{LS}} = 0$$

$G_{LS} - G_{HS} \gg RT$ のとき $n_{LS} \to 0$（HS が安定）であり、$G_{HS} - G_{LS} \gg RT$ のとき $n_{LS} \to 1$（LS が安定）である。$g_{HS}/g_{LS} = 5$ の比率は、クロスオーバーを $RT\ln(5)$ に比例する量だけ高圧側へシフトさせる。$G_{HS}$ と $G_{LS}$ は圧力と温度の複雑な非線形関数であるため、コード中ではこの平衡条件を解析的にではなくグリッドサーチにより数値的に解いている。

### 22.3.4 線形混合による有効体積

混合スピン鉱物の有効モル体積は、HS と LS の体積をスピン分率で重み付けした線形内挿である：

$$V_{\text{eff}} = n_{LS} \, V_{LS} + (1 - n_{LS}) \, V_{HS}$$

$V_{LS} < V_{HS}$（LS $\text{Fe}^{2+}$ は約 $8\%$ 小さい）であるため、$n_{LS}$ の増加は通常の圧縮を超える体積減少を生じさせる。同様の線形混合が $K_S$、$G_S$、密度にも適用され、有効弾性物性が得られる。`SpinCrossoverCalculator.ComputeSpinState` に直接実装されている。

### 22.3.5 LS 端成分パラメータのスケーリング

LS 端成分は第一原理計算に基づく近似的スケーリングにより HS 端成分から生成される：

$$V_{0,LS} = 0.92 \, V_{0,HS}, \quad K_{0,LS} = 1.15 \, K_{0,HS}, \quad G_{0,LS} = 0.95 \, G_{0,HS}, \quad \theta_{0,LS} = 0.95 \, \theta_{0,HS}$$

$8\%$ の体積減少は LS $\text{Fe}^{2+}$ のより小さなイオン半径を、$15\%$ の体積弾性率増加はよりコンパクトで硬い LS 状態を反映する。スピン量子数は 0 に設定され、磁気エントロピーは消去される。`SpinCrossoverCalculator.CreateLSEndmember` に実装されている。

### 22.3.6 混合スピン弾性物性からの有効地震波速度

有効弾性率と密度から地震波速度が計算される：

$$V_p = 1000\sqrt{\frac{K_{S,\text{eff}} + \frac{4}{3}G_{S,\text{eff}}}{\rho_{\text{eff}}}}, \qquad V_s = 1000\sqrt{\frac{G_{S,\text{eff}}}{\rho_{\text{eff}}}}$$

ここで 1000 の係数は GPa と g/cm$^3$ 単位から m/s への単位変換に対応する。クロスオーバー領域では、$K_{S,\text{eff}}$ の軟化が $V_p$ を純粋 HS モデルに対して低下させ、地震学的に観測可能な速度異常を生じさせる可能性がある。

## 22.4 物理概念

### 22.4.1 鉄スピンクロスオーバー（HS-LS 遷移）

**定義**：圧力誘起の電子遷移であり、$\text{Fe}^{2+}$ の 3d 電子が高スピン配置（$S=2$、$t_{2g}$ と $e_g$ に分布した 4 個の不対電子）から低スピン配置（$S=0$、3 個の $t_{2g}$ 軌道にすべて 6 電子が対をなして収容）へと再配列する。結晶場分裂が電子対形成エネルギーを超えるにつれて、圧力の増加とともに生じる。

**物理的直感**：低圧では電子は反発を最小化するために全 5 つの d 軌道に広がることを好む（Hund の法則）。これは空いた部屋に人が分散するようなものである。圧力が結晶を圧縮すると、上位 $e_g$ 軌道を占有するためのエネルギーペナルティが劇的に増大する。最終的に、下位 $t_{2g}$ 軌道で電子を対にする方がエネルギー的に有利になる。この遷移が漸進的であるのは、熱的揺らぎにより個々の Fe 原子がどちらの状態にも存在でき、系が全自由エネルギーを最小化する最適混合状態を見出すためである。

**よくある誤解**：

- スピンクロスオーバーを急激な相転移として扱うこと。マントル温度では、格子サイト上の HS・LS 集団の配置エントロピーにより数十 GPa にわたる広いクロスオーバーとなる。
- スピンクロスオーバーが体積のみに影響すると仮定すること。体積弾性率、せん断弾性率、Debye 温度も変化し、地震波速度に影響する異常な弾性軟化を生じさせる。
- スピンクロスオーバーを構造相転移と混同すること。フェロペリクレースの結晶構造（岩塩型、B1）はスピンクロスオーバーの間に変化しない。変化するのは $\text{Fe}^{2+}$ イオンの電子状態のみである。

### 22.4.2 混合スピン領域における弾性軟化

**定義**：HS と LS 状態が共存する圧力範囲において、鉱物の有効体積弾性率 $K_S$ はいずれの純粋スピン状態よりも低下する。圧力の増加に伴う HS から LS への漸進的変換が、通常の格子圧縮を超える追加的な体積減少メカニズムとなるためである。

**物理的直感**：2 種類の異なる剛性のバネを含む材料の圧縮を想像してほしい。クロスオーバー領域では、より多くの圧力を加えても既存のバネを圧縮するだけでなく、硬い長いバネ（HS）をさらに硬いが短いバネ（LS）へと変換する。この変換自体が体積を吸収し、材料をいずれのバネ単独よりも柔らかく見せる。

**よくある誤解**：

- 軟化が材料の力学的不安定性を意味すると信じること。体積弾性率は低下するが正のままであり、単に予想より圧縮しやすいだけである。
- $K_S$ の線形混合が完全な異常を捕捉すると仮定すること。$n_{LS}$ が圧力とともに変化する自己無撞着な熱力学的処理からの真の軟化は、固定端成分値間の線形内挿よりもさらに顕著になりうる。
- 軟化がすべての弾性率に等しく影響すると考えること。体積弾性率は体積変化に結合するため最も影響を受け、せん断弾性率の異常はより小さい。

### 22.4.3 スピン状態の Boltzmann 分布

**定義**：HS と LS の鉄原子の平衡分率は、二状態間の Gibbs エネルギー差と電子縮退度に依存する Boltzmann 型分布により決定される。$n_{LS}/n_{HS}$ の比は $(g_{LS}/g_{HS}) \cdot \exp(-(G_{LS} - G_{HS})/RT)$ に比例する。

**物理的直感**：格子サイト上の各 Fe 原子は、エネルギーコスト（Gibbs エネルギー差）と各状態が実現される方法の数（電子縮退度）に基づいてどちらのスピン状態を採用するかを独立に決定する。HS 状態は 5 つの縮退スピン配向を持つのに対し LS は 1 つのみであるため、HS 状態には 5:1 の統計的優位がある。これが、エントロピーの役割を増幅する高温がクロスオーバーをより高圧へシフトさせる理由である。

**よくある誤解**：

- 電子縮退度を含めずに単純な二準位 Boltzmann モデルを適用すること。$g_{HS}/g_{LS}$ 因子を省略すると、マントル温度でクロスオーバー圧力を数十 GPa 過小評価する。
- Gibbs エネルギーが圧力の単純な線形関数であると扱うこと。$G_{HS}$ と $G_{LS}$ は完全な EOS から計算される複雑な非線形関数である。
- 二状態モデルが厳密であると仮定すること。実際には中間スピン状態、スピン-軌道結合、鉄-鉄相互作用がこの描像を修正しうるが、二状態モデルは支配的な物理を捕捉する。

### 22.4.4 LS 端成分のパラメータ化

**定義**：LS 端成分は常温常圧では独立に測定されない（LS 状態は低圧では準安定であるため）。代わりに、状態方程式パラメータは HS パラメータのスケーリングにより近似される：$V_0$ は約 $8\%$ 減少、$K_0$ は約 $15\%$ 増加、$G_0$ は約 $5\%$ 減少、Debye 温度は約 $5\%$ 減少。スピン量子数は 0 に設定され、すべての磁気エントロピーが消去される。

**物理的直感**：LS $\text{Fe}^{2+}$ イオンはすべての d 電子がコンパクトな $t_{2g}$ 軌道にあるため、物理的に小さく（低い $V_0$）、圧縮しにくい（高い $K_0$）。これらのスケーリング因子は、任意の条件で LS 状態にアクセスできる第一原理計算から得られ、常温常圧では直接実験的に測定できない参照パラメータを提供する。

**よくある誤解**：

- スケーリング因子が正確に既知であると仮定すること。これらは研究間で異なる理論計算に基づく近似値である。
- LS 端成分が別の鉱物として存在すると信じること。同一鉱物の異なる電子配置であり、結晶構造は変化しない。
- 同じスケーリング因子をすべての鉱物に適用すること。実際の変化は結晶構造と鉄サイトの幾何学的配置に依存する。

### 22.4.5 スピンクロスオーバーの地震学的含意

**定義**：フェロペリクレースにおけるスピンクロスオーバーによる弾性軟化と密度異常は、中部下部マントル（深さ約 1000--2100 km）の地震波速度に影響を与えうる。

**物理的直感**：地震波速度は弾性率と密度に依存する。クロスオーバー領域では体積弾性率の軟化が $V_p$ を低下させる一方、密度の増加は逆の効果を持つ。正味の結果は鉄含有量、温度、集合体中のフェロペリクレースの割合に依存する。約 $20\%$ のフェロペリクレース（15--20 mol% FeO 含有）を持つパイロライト的マントルでは、集合的 $V_p$ と $V_s$ への効果は 1--2% 程度と推定される。

**よくある誤解**：

- スピンクロスオーバーから鋭い地震学的不連続面を期待すること。マントル温度では遷移は 30--50 GPa にわたり緩やかすぎて反射面を生じない。速度勾配の異常として現れる。
- この効果が LLSVP のような主要地震構造を説明するのに十分大きいと仮定すること。集合的速度への効果は微妙であり、大きな横方向速度変動は組成または熱的不均質を必要とする。
- スピンクロスオーバーが鉄分配にも影響することを無視すること。スピン状態の変化に伴う鉄の化学ポテンシャルの変化がフェロペリクレースとブリッジマナイト間の鉄分布を修正する。

## 22.5 計算例

### 22.5.1 60 GPa, 2000 K におけるウスタイトの平衡低スピン分率

**問題**：`SpinCrossoverCalculator` を用いて、ウスタイト（FeO）の 60 GPa, 2000 K における平衡低スピン分率 $n_{LS}$、有効体積、体積弾性率、地震波速度を計算せよ。

**解法**：

1. SLB2011 データベースからウスタイト ('wu') の HS 端成分を読み込む。
2. `CreateLSEndmember` により LS 端成分を生成する（$V_{0,LS} = 0.92 \times V_{0,HS}$、$K_{0,LS} = 1.15 \times K_{0,HS}$、$S=0$）。
3. `MieGruneisenEOSOptimizer` を用いて両スピン状態の EOS を 60 GPa, 2000 K で独立に解く。
4. $n_{LS}$ に対する $F_{\text{mix}}$ のグリッドサーチにより平衡スピン分率を求める。
5. 線形混合により有効物性を計算する。

**数値パラメータ**：

| パラメータ | 値 |
|---|---|
| $P$ | 60 GPa |
| $T$ | 2000 K |
| $V_{0,HS}$ (wu) | 12.264 cm$^3$/mol（SLB2011 より）|
| $V_{0,LS}$ | $12.264 \times 0.92 = 11.283$ cm$^3$/mol |
| $K_{0,HS}$ (wu) | 149 GPa |
| $K_{0,LS}$ | $149 \times 1.15 = 171.35$ GPa |
| $g_{HS}$ | 5 |
| $g_{LS}$ | 1 |

**結果**：60 GPa, 2000 K はクロスオーバー領域の内部にあり、$n_{LS}$ は中間値（0.3--0.9 の範囲）をとると期待される。テストスイート（`SpinCrossoverTests.LSFraction_At60GPa_InCrossoverRegion`）はこの条件での $n_{LS}$ が $[0.3, 0.9]$ に収まることを検証している。

**解釈**：HS と LS 状態が共存するため、鉱物の物性は二つの異なる電子状態の加重平均となる。有効体積は純粋 HS に比べて減少し、体積弾性率は HS と LS 端成分値の中間にある。

### 22.5.2 スピンクロスオーバーの温度依存性

**問題**：60 GPa において $T = 300$ K、1500 K、3000 K での $n_{LS}$ を計算し、磁気エントロピーの議論を用いてその傾向を説明せよ。

**解法**：圧力を 60 GPa に固定し、`SpinCrossoverCalculator.ComputeSpinState` を 3 つの温度で実行する。

**数値パラメータと期待される結果**：

- $T = 300$ K：$n_{LS}$ は高い値（1 に近い）。低温での急峻なクロスオーバー。
- $T = 1500$ K：$n_{LS}$ は中間値。クロスオーバーはブロード化。
- $T = 3000$ K：$n_{LS}$ はより低い値（HS が増加）。クロスオーバーはさらに高圧へシフト。
- 磁気エントロピー差：$R\ln(5) - R\ln(1) = 13.38$ J/mol/K（Fe 原子あたり）
- 3000 K でのエントロピー安定化：$3000 \times 13.38 = 40.1$ kJ/mol（HS を安定化）

**解釈**：300 K では熱エネルギーが小さいため Gibbs エネルギー差のみがスピン状態を制御し、60 GPa ではほぼ LS の急峻なクロスオーバーを生じる。3000 K では磁気エントロピー寄与 $T R\ln(5) = 40.1$ kJ/mol が HS 状態を強く安定化し、クロスオーバーの中点を 60 GPa よりはるかに高圧にシフトさせる。これは、室温実験よりもマントル温度条件においてスピンクロスオーバーが有意に高い圧力で生じ、はるかにブロードであることを示す。

### 22.5.3 クロスオーバー領域における体積弾性率異常

**問題**：60 GPa, 2000 K においてスピンクロスオーバー計算による $K_{S,\text{eff}}$ と純粋 HS 体積弾性率を比較し、弾性軟化の大きさを推定せよ。

**解法**：

1. `MieGruneisenEOSOptimizer` により 60 GPa, 2000 K でのウスタイトの純粋 HS 物性を計算する。
2. `SpinCrossoverCalculator` によりスピンクロスオーバーの有効物性を計算する。
3. $K_S$ の値を比較し、分数的低下を算出する。

**解釈**：クロスオーバー領域における有効 $K_S$ は純粋 HS 値から測定可能な差を示す。テストスイート（`SpinCrossoverTests.KSAnomaly_InCrossoverRegion`）は、$K_{S,\text{eff}}$ が純粋 HS と LS の端成分値の間にあり、純粋 HS の $K_S$ から少なくとも 1 GPa 異なることを検証している。$n_{LS}$ の圧力依存性（$dn_{LS}/dP$）からの追加的軟化を含む自己無撞着な処理では、さらに大きな異常が生じうる。約 $20\%$ のフェロペリクレースを含む下部マントル集合体では、これは控えめだが潜在的に地震学的に検出可能な速度異常に対応する。

## 22.6 計算手法

`SpinCrossoverCalculator` はスピンクロスオーバー計算を 3 つの主要段階で実装する。

**第一段階：LS 端成分の生成**

`CreateLSEndmember` は HS 端成分にスケーリング因子を適用する：$V_0 \times 0.92$、$K_0 \times 1.15$、$G_0 \times 0.95$、$\theta_{\text{Debye}} \times 0.95$。`SpinQuantumNumber` は 0 に設定し磁気エントロピーを消去する。その他のパラメータ（$K_1'$、$K_2'$、$G_1'$、$G_2'$、$\gamma_0$、$q_0$、$\eta_{S0}$、$F_0$、Landau パラメータ）は変更なくコピーされる。

**第二段階：両スピン状態の EOS 解法**

`ComputeSpinState` メソッドは `MieGruneisenEOSOptimizer.ExecOptimize()` を用いて HS と LS 端成分のそれぞれについて指定 $P$, $T$ での状態方程式を独立に解く。各端成分について Gibbs エネルギー $G_{HS}$ と $G_{LS}$ を含む完全な熱力学物性が得られる。EOS ソルバーは固定 $P$, $T$ で $G = F + PV$ を最小化する平衡体積を求め、Birch-Murnaghan + Mie-Gruneisen-Debye 方程式の反復的解法を行う。

**第三段階：平衡スピン分率の決定**

$F_{\text{mix}}(n_{LS})$ を $[0, 1]$ の範囲で最小化する。二段階グリッドサーチを使用する：

1. **粗いグリッド**：$[0, 1]$ にわたる 1001 点の初期グリッド。
2. **精密グリッド**：粗い最小値の周囲 $\pm 0.002$ の窓内に 1001 点。

これにより $n_{LS}$ に対して約 $4 \times 10^{-6}$ の有効分解能が達成される。グリッドサーチは勾配法が $n_{LS} = 0$ および $n_{LS} = 1$ 近傍の対数的配置エントロピー項の発散により遭遇しうる収束問題を回避するため、ロバストである。境界ケースは `ComputeFMix` 内で明示的に処理される：$n_{LS} < 10^{-12}$ のとき $(1-n)\ln(1/g_{HS})$ 項のみが寄与し、$n_{LS} > 1 - 10^{-12}$ のとき $n\ln(1/g_{LS})$ 項のみが寄与する。

**有効物性の計算**

有効物性は線形内挿により計算される：$V_{\text{eff}}$、$K_{S,\text{eff}}$、$G_{S,\text{eff}}$、密度はスピン分率で重み付けされた HS と LS 値の間で線形混合される。地震波速度は有効弾性率と密度から標準公式（$V_p = 1000\sqrt{(K_S + 4G/3)/\rho}$、$V_s = 1000\sqrt{G/\rho}$）により計算される。

**計算コスト**

計算コストは 2 回の `MieGruneisenEOSOptimizer` 呼び出し（HS 用と LS 用、各々反復的体積最適化を含む）が支配的である。グリッドサーチは無視できるコスト（2002 回の単純代数式評価）を追加する。$P$, $T$ 点あたりの総壁時間は典型的に数ミリ秒であり、クロスオーバープロファイルのプロットのための圧力・温度範囲のスイープが実行可能である。

## 22.7 コード対応

本章の理論計算は以下のクラスとメソッドに対応する。

| 理論的概念 | クラス/メソッド | 説明 |
|---|---|---|
| LS 端成分生成 | `SpinCrossoverCalculator.CreateLSEndmember` | HS パラメータのスケーリング |
| スピン状態計算 | `SpinCrossoverCalculator.ComputeSpinState` | 両端成分の EOS 解法と $n_{LS}$ 決定 |
| 混合自由エネルギー | `SpinCrossoverCalculator.ComputeFMix` | $F_{\text{mix}}(n_{LS})$ の評価 |
| EOS 解法 | `MieGruneisenEOSOptimizer.ExecOptimize` | 各端成分の $P$-$T$ 物性計算 |
| HS 端成分パラメータ | `SLB2011Endmembers` ('wu') | ウスタイトの EOS パラメータ |
| GUI インターフェース | `SpinCrossoverViewModel` | 鉱物選択、$P$-$T$ 指定、結果表示 |

典型的な使用フローは以下の通りである：

1. `SLB2011Endmembers` からウスタイト ('wu') の HS 端成分パラメータを取得する。
2. `CreateLSEndmember` により近似的 LS 端成分を生成する。
3. `ComputeSpinState` を指定 $P$, $T$ で呼び出す。
4. 返された `SpinCrossoverResult` から $n_{LS}$、有効 $V$, $K_S$, $G_S$, $V_p$, $V_s$ を得る。

テストスイートでは以下の検証が行われている：

- 低圧（例：0.1 GPa）での $n_{LS} < 0.01$（HS 支配）
- 60 GPa, 2000 K での $n_{LS} \in [0.3, 0.9]$（クロスオーバー領域）
- 高圧での $n_{LS} > 0.99$（LS 支配）
- クロスオーバー領域での $K_{S,\text{eff}}$ が HS と LS 端成分値の間にあること

## 22.8 歴史

高圧における遷移金属酸化物の電子スピン状態の研究は、1950--60 年代に Orgel、Tanabe、Sugano らにより発展した結晶場理論に根を持つ。圧力がスピン遷移を誘起しうることは、圧縮が結晶場分裂を増大させるという理解に基づいて、実験的に観測されるより何十年も前に理論的に予測されていた。

しかし、下部マントル鉱物における最初の直接実験的観測は 2003 年に訪れた。Badro らはシンクロトロン X 線発光分光法を用いてフェロペリクレースにおける鉄のスピン崩壊を中部下部マントルに対応する圧力で検出した。この発見はスピンクロスオーバー鉱物物理学の分野を切り開き、後続のすべての理論的・実験的研究を動機づけた。

続く数年間で、Lin ら（2005）と Sturhahn ら（2005）がクロスオーバーの熱力学的枠組みを確立し、電子縮退因子を含む Boltzmann 統計が有限温度での遷移の漸進的性質を説明できることを示した。Tsuchiya ら（2006）はマグネシオウスタイトにおけるクロスオーバーの最初の第一原理計算を提供し、強い温度依存性と磁気エントロピーの役割を実証した。Wentzcovitch ら（2009）はこれを異常圧縮率の詳細な計算へ拡張し、クロスオーバー領域における弾性軟化の最初の定量的推定を提供した。

スピンクロスオーバーを組み込んだ自己無撞着熱力学モデルの開発は継続的な努力である。SLB2011 の枠組みは HS 端成分の状態方程式インフラを提供するが、ネイティブにはスピンクロスオーバーを含まない。本コードベースの `SpinCrossoverCalculator` は近似的 LS 端成分を生成し平衡混合を計算することでこの枠組みを拡張する。Fukui ら（2012）や Wu ら（2013）によるより洗練されたアプローチは、二端成分近似を回避してスピンクロスオーバーを分配関数に直接組み込む。

スピンクロスオーバーが地震学的に検出可能かどうかは活発に議論されている。Marquardt らの研究が関連条件でのますます精密な弾性測定を提供している。スピンクロスオーバーと鉄分配の相互作用は、自己無撞着にモデル化され始めたばかりの複雑さの層をさらに追加する。

## 22.9 未解決課題

1. **LS 端成分の正確な EOS パラメータ**：`CreateLSEndmember` で使用される近似スケーリング因子（$V_0 \times 0.92$, $K_0 \times 1.15$）は第一原理研究間で異なり、小さな変化がクロスオーバー圧力を数 GPa シフトさせうる。

2. **ブリッジマナイトにおけるスピンクロスオーバー**：$\text{Fe}^{2+}$ がより大きく歪んだサイトを占めるブリッジマナイトでは、クロスオーバー圧力とパラメータスケーリングがフェロペリクレースと有意に異なりうる。B サイトの $\text{Fe}^{3+}$ はさらなる複雑さを加える。

3. **弾性物性の線形混合の妥当性**：クロスオーバー領域における有効 $K_S$ と $G_S$ の計算に線形混合は十分か、あるいは $dn_{LS}/dP$ の寄与を考慮した自己無撞着熱力学的処理が有意に異なる結果を与えるか。

4. **鉄-鉄相互作用の効果**：現行モデルは各 Fe サイトを独立に扱うが、現実的な鉄濃度（15--20 mol% FeO）でのスピン-スピン結合が協同的効果や一次転移的挙動を生じさせる可能性がある。

5. **自己無撞着下部マントルモデル**：スピンクロスオーバーが鉄分配に影響し、鉄含有量がクロスオーバー圧力をシフトさせるというフィードバックループは未だ自己無撞着に解決されていない。

6. **地震学的検出可能性**：中部下部マントルにおける予測速度異常は微妙（集合的岩石で約 1--2%）であり、熱的・組成的効果と区別するには非常に精密な地震データと参照モデルが必要である。

7. **スピン-軌道結合の効果**：単純な $2S+1$ 多重度は電子構造を完全には捕捉しない可能性がある。特に $t_{2g}^5$ HS 状態では軌道縮退が追加的エントロピーに寄与しうる。

## 22.10 演習問題

**基礎**

1. `SpinCrossoverCalculator` を用いて、ウスタイトの $n_{LS}$ を $T = 2000$ K で 0--120 GPa の圧力範囲にわたって計算せよ。$n_{LS}(P)$ をプロットし、クロスオーバー圧力（$n_{LS} = 0.5$）を特定せよ。Lin ら（2013）の実験範囲（約 40--70 GPa）と比較せよ。

2. 上記の計算を $T = 300$ K、1500 K、3000 K で繰り返し、4 つの $n_{LS}(P)$ 曲線を同一軸上にプロットせよ。高温がクロスオーバーをより高圧へシフトさせブロード化させることを実証し、磁気エントロピーの議論を用いてその傾向を説明せよ。

3. 2000 K のスピンクロスオーバーにわたる有効体積弾性率 $K_{S,\text{eff}}(P)$ を計算し、純粋 HS の $K_S(P)$ 曲線と比較せよ。最大軟化（$K_S$ の分数的低下）とそれが生じる圧力を定量化せよ。

**中級**

4. $F_{\text{mix}}$ の最小化から $dn_{LS}/dP$ を解析的に導出し、線形混合近似を超える有効圧縮率への追加項を示せ。クロスオーバー中点でのこの補正の大きさを推定せよ。

5. `CreateLSEndmember` を修正して LS パラメータスケーリングに対するクロスオーバー圧力の感度をテストせよ：$V_0$ 減少 $6\%$、$8\%$、$10\%$ および $K_0$ 増加 $10\%$、$15\%$、$20\%$ を試せ。結果の $n_{LS}(P)$ 曲線をプロットし、クロスオーバー圧力がどのパラメータに最も敏感かを議論せよ。

6. $80\%$ ブリッジマナイト + $20\%$ フェロペリクレース $(\text{Mg}_{0.85}\text{Fe}_{0.15})\text{O}$ の簡略化された下部マントル集合体について、60 GPa, 2000 K での集合的 $V_p$ に対するスピンクロスオーバーの効果を推定せよ。スピンクロスオーバー補正の有無で集合的速度を比較せよ。

**発展**

7. ウスタイトの $G_{HS}(P)$ と $G_{LS}(P)$ の Gibbs エネルギー曲線を 2000 K で 0--120 GPa にわたって計算せよ。両曲線を同一軸上にプロットし、交差するクロスオーバー圧力を特定せよ。混合状態の自由エネルギー $F_{\text{mix}}$ もプロットし、クロスオーバー領域で両純粋状態を下回ることを示せ。

8. クロスオーバー中点（$n_{LS} = 0.5$）での配置エントロピー $S_{\text{config}}$ を、電子縮退因子の有無（$g_{HS}=5$, $g_{LS}=1$ 対 $g_{HS}=g_{LS}=1$）で比較せよ。縮退を含めることが平衡をシフトさせることを示し、クロスオーバー圧力の差を定量化せよ。

| 関連章 | 関連の性質 | 説明 |
|---|---|---|
| 第9章（磁気的寄与）| 前提 | 磁気自由エネルギー $F_{\text{mag}} = -TrR\ln(2S+1)$ が HS ($S=2$, $g=5$) と LS ($S=0$, $g=1$) 状態間のエントロピー差を提供し、クロスオーバーの温度依存性を駆動する。 |
| 第6章（Mie-Gruneisen EOS ソルバー）| 前提 | `SpinCrossoverCalculator` は `MieGruneisenEOSOptimizer` に依存して各 $P$, $T$ 条件で HS・LS 両端成分の EOS を独立に解く。 |
| 第4章（弾性率と地震波速度）| 前提 | 有効弾性率から地震波速度への変換は第4章の標準的速度表式に基づく。 |
| 第7章（SLB2011 鉱物データベース）| 応用 | スピンクロスオーバー計算への入力となる HS ウスタイト端成分は SLB2011 データベースから得られる。 |
| 第23章（鉄分配）| 発展 | スピンクロスオーバーがフェロペリクレース中の鉄の Gibbs エネルギーを修正し、フェロペリクレース・ブリッジマナイト間の平衡鉄分配に影響する。 |
| 第10章（弾性混合モデル）| 応用 | HS-LS 端成分間の弾性物性の線形混合は弾性混合モデルの特定の適用例である。 |
| 第15章（PREM と地球モデル）| 応用 | スピンクロスオーバーは中部下部マントルの予測速度プロファイルに影響する。PREM との比較が鉄含有量と温度を制約しうる。 |
| 第20章（LLSVP と ULVZ）| 応用 | 深部マントル地震構造の解釈において、スピンクロスオーバーの速度効果を組成・熱的不均質と区別する必要がある。 |

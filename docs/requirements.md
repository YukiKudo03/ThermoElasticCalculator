## 1. プロジェクト目的

高温高圧条件下のマントル鉱物の弾性的性質を計算し、それらの混合モデルから岩石の物性を算出するクロスプラットフォーム デスクトップソフトウェアを開発する。

### 1.1 背景

地球内部の弾性波速度構造の解釈には、高温高圧条件下における鉱物の物性（弾性波速度、密度、弾性率等）の知識が不可欠である。本ソフトウェアは、Stixrude & Lithgow-Bertelloni (2011) の熱力学モデルに基づき、実験パラメータから鉱物の物性を自己無撞着に計算する。

### 1.2 対象ユーザー

- 地球物理学・鉱物物理学の研究者
- マントルの組成・構造に関心のある地球科学者

5. 既存の WinForms 版（thermo-dynamics/）は非推奨
6. Debye関数は Simpson 積分（500分点）で計算。極端な高温条件（T > 2500K）で一部鉱物の精度が低下する場合がある
7. BM3近似の適用限界：低対称鉱物（quartz, nepheline等）の高圧相転移域近傍では精度低下

## 6. 用語定義

| 用語 | 定義 |
|------|------|
| EOS | Equation of State（状態方程式） |
| BM3 | Birch-Murnaghan 3rd order（3次Birch-Murnaghan） |
| Gruneisen パラメータ | 熱膨張と格子振動の関係を記述するパラメータ γ |
| Debye モデル | 固体の格子振動を連続弾性体で近似するモデル |
| Voigt 平均 | 等歪み仮定による弾性率の上界推定 |
| Reuss 平均 | 等応力仮定による弾性率の下界推定 |
| Hill 平均 | Voigt と Reuss の算術平均（VRH平均） |
| Hashin-Shtrikman | 変分原理に基づく弾性率の上下界推定 |
| Finite strain | Eulerian有限歪み f = [(V₀/V)^(2/3) - 1] / 2 |
| MVVM | Model-View-ViewModel パターン |
| Avalonia | クロスプラットフォーム .NET UI フレームワーク |
| Landau 相転移 | 変位型構造相転移（α-βクオーツ等）のモデル |
| 固溶体 | 端成分鉱物の混合物。配置エントロピーと過剰自由エネルギーで記述 |
| Van Laar モデル | 非対称固溶体の過剰自由エネルギー計算モデル |
| Gibbs 最小化 | 全相のGibbs自由エネルギー合計を最小化し安定相集合を決定する手法 |
| BurnMan | SLB2011と同一フレームワークを実装したPython参照ライブラリ |
| SLB2011 | Stixrude & Lithgow-Bertelloni (2011) の熱力学パラメータセット |
